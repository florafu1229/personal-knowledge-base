双指针
    移动零: 双指针都从左边出发，慢指针记录非零位置，快指针找非零元素。
    盛最多水的容器：双指针从两端向中间收缩，每次移动高度较小的指针，记录最大容量。
    三数之和：先排序，外层遍历第一个数，内层用双指针从两端向中间收缩，找两数之和等于负的第一个数。 注意要排除很多条件
滑动窗口（同向向右的双指针）
    无重复字符的最长子串: 用哈希表记录字符位置，遇到重复时移动左指针跳过重复字符。
    找到字符串中所有字母异位词: 两个26字母字典，向右滑动固定窗口， 检查两个字典是否相同
子串
    和为K的子数组：用前缀和 + 哈希表存储，查找 prefix_sum - k 是否存在，存在则说明中间子数组和为 k。
普通数组
    最大子数组之和: 动态规划，current_sum = max(nums[i], current_sum + nums[i])
    合并区间:先按起点排序.sort(key=lambda x: x[0])，遍历时如果当前区间与上一个重叠就合并，否则加入结果。
    轮转数组: 三次翻转：全翻、前 k 个翻、剩余翻，翻转方法用双向指针交换实现
    除了自身以外数组的乘积: 从左向右算前缀积，然后从右向左算后缀积
矩阵
    矩阵置零: 第一次循环：用第一行和第一列作为标记空间，原地标记哪些行/列需要置零。第二次循环将里面的元素按标记置为0， 最后处理第一行和第一列
    螺旋矩阵: 用四个边界模拟螺旋，按上→右→下→左顺序遍历，每遍历完一边收缩边界。循环条件是：上边界小于下，左小于右
    旋转图像: 先转置， 再水平翻转(matrix[i].reverse())
    搜索二维矩阵II:从右上角开始，大于 target 往左，小于 target 往下，最多走 m+n 步。

    环形链表: 快慢指针，快指针走两步，慢指针走一步，有环必相遇，无环快指针先到末尾。

    合并两个有序链表:用哨兵节点，比较两个链表当前节点，小的接上，指针后移，最后接上剩余部分。
    两数相加: 从低位到高位逐位相加，处理进位，用哨兵节点构建结果链表。
    删除链表的倒数第N个节点: 快指针先走 n 步，然后快慢指针同时走，快指针到末尾时，慢指针的下一个就是要删除的节点。
    两两交换链表中的节点: 用哨兵节点，每次交换两个节点的指针关系，然后移动到下一对。
    随机链表的复制 : 第一遍建立老新映射，第二遍根据映射设置 next 和 random 指针。

动态规划
    爬楼梯: dp[i] = dp[i-1] + dp[i-2]，每次爬 1 或 2 步。
    杨辉三角: 每个数等于上方两数之和，dp[i][j] = dp[i-1][j-1] + dp[i-1][j]。
    打家劫舍: dp[i] = max(dp[i-1], dp[i-2] + nums[i])，选或不选当前房屋。
    完全平方数: dp[i] = min(dp[i - j*j] + 1)，尝试所有可能的平方数。
    零钱兑换: dp[i] = min(dp[i - coin] + 1)，尝试所有硬币。
    单词拆分: dp[i] 表示前 i 个字符能否被拆分，dp[i] = any(dp[j] and s[j:i] in wordDict)。
    最长递增子序列: dp[i] = max(dp[j] + 1) for j < i and nums[j] < nums[i]。
    乘积最大子数组: 维护当前最大和最小乘积，dp[i] = max(nums[i], max_dp[i-1] * nums[i], min_dp[i-1] * nums[i])。
    分割等和子集: 转换为 0-1 背包问题，dp[j] = dp[j] or dp[j - nums[i]]。
多维动态规划
    不同路径: dp[i][j] = dp[i-1][j] + dp[i][j-1]，从上方或左方到达。
    最小路径和: dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])，选择最小路径。
    //最长回文子串: 中心扩展法，以每个字符为中心向两边扩展，或用 dp[i][j] 表示 s[i:j+1] 是否回文。
    最长公共子序列： dp[i][j] = dp[i-1][j-1] + 1 if s1[i-1] == s2[j-1] else max(dp[i-1][j], dp[i][j-1])。
    编辑距离： dp[i][j] = dp[i-1][j-1] if 相同 else min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1。

技巧
    只出现一次的数字 : 用异或运算，相同数字异或为 0，0 异或任何数为该数，最终得到只出现一次的数字。
    多数元素 : 摩尔投票法：维护候选数和计数，相同加 1，不同减 1，计数为 0 更换候选，最后剩下的就是多数元素。
    颜色分类：三指针：left 指向 0，right 指向 2，curr 遍历，遇到 0 换到 left，遇到 2 换到 right。

    寻找重复数： 快慢指针：数组索引→值形成链表，重复数字导致环，环入口就是重复数。

    买卖股票的最佳时机: 贪心：维护最小买入价格，每天计算卖出的利润，取最大值。

回溯: 回溯算法是一种通过尝试所有可能来解决问题的策略：当走到死路时，退回上一步，换条路继续尝试。
    全排列： 回溯：每次选一个未用数字，递归处理剩余，完成后撤销标记，直到选完所有数字。
    子集 ： 每个元素选或不选，递归构建所有组合，每次递归都保存当前路径。
    电话号码的字母组合: 每个数字遍历对应字母，递归处理下一个数字，到达末尾时保存组合。
    组合总和：从当前位置开始，选每个数字递归处理剩余目标，允许重复使用数字，避免重复组合。
    括号生成 ： 左括号数 < n 时加左括号，右括号数 < 左括号数时加右括号，直到长度为 2n。
    单词搜索 ： 回溯+DFS：从每个位置开始，四个方向搜索匹配，标记已访问单元格，匹配完或失败后恢复。
    分割回文串 ： 回溯：遍历所有分割点，如果子串是回文则继续递归，到达末尾时保存分割方案。
